# 技能：compress-memory

提取小说前情提要，防止上下文窗口超载。

## 执行流程

禁止合并步骤执行，先列出todos，执行完成标记完成

- [ ] step1: 验证项目结构
- [ ] step2: 检测是否为分卷模式
- [ ] step3: 扫描章节
  - **分卷模式**：扫描 `04_Chapters/volume_*/` 下的所有章节，按卷和章节号排序
  - **非分卷模式**：扫描 `04_Chapters/` 下的所有章节，按章节号排序
- [ ] step4: 确定未总结章节
  - 对比 `03_Outline/summary.md` 记录，找出未总结章节
  - `03_Outline/summary.md` 为空：总结所有章节
  - `03_Outline/summary.md` 有记录：总结未记录章节
- [ ] step5: 提炼核心摘要
  - 阅读未总结的章节文本
  - 提取关键剧情推进、角色关系变化、伏笔悬念、重要道具/能力得失
- [ ] step6: 保存摘要
  - 追加到 `03_Outline/summary.md`
  - **分卷模式**：按卷组织，添加卷标题
  - **非分卷模式**：按章节号顺序追加
  - 包含章节号和标题（如有）
- [ ] step7: 报告结果
  - 列出被总结章节（分卷模式注明卷号）
  - 提示可参考summary.md而非完整章节

## 错误处理

- `04_Chapters/` 为空/不存在：通知并停止
- 文件读取失败：跳过并记录警告
- `summary.md` 写入失败：尝试备份或通知手动保存
- 无法解析章节编号：按修改时间排序
