# 技能：compress-memory

提取小说前情提要，防止上下文窗口超载。

## 执行流程：

**验证项目结构**：检查文件`03_Outline/summary.md`是否存在，不存在则创建。

**检测分卷模式**：检查 `03_Outline/volume_<卷号>_detailed.md` 是否存在。

- 如果存在，进入分卷模式
- 如果不存在，进入兼容模式

1. **扫描章节**：
   - **分卷模式**：扫描 `04_Chapters/volume_*/` 下的所有章节，按卷和章节号排序
   - **兼容模式**：扫描 `04_Chapters/` 下的所有章节，按章节号排序
2. **确定未总结章节**：对比 `03_Outline/summary.md` 记录，找出未总结章节。
   - summary.md为空：总结所有章节
   - 有记录：从最后总结章节后开始
   - **分卷模式**：按卷处理
3. 仔细阅读这些未总结的章节文本。
4. 提炼核心摘要：
   - 关键剧情推进
   - 角色关系变化
   - 伏笔悬念
   - 重要道具/能力得失
5. **保存摘要**：追加到 `03_Outline/summary.md`。
   - **分卷模式**：按卷组织，添加卷标题
   - **兼容模式**：按章节号顺序追加
   - 包含章节号和标题（如有）
6. **报告结果**：向用户报告压缩结果。
   - 列出被总结章节（分卷模式注明卷号）
   - 估计释放阅读负担（字数/token估算）
   - 提示可参考summary.md而非完整章节

## 错误处理

- `04_Chapters/` 为空/不存在：通知并停止
- 文件读取失败：跳过并记录警告
- `summary.md` 写入失败：尝试备份或通知手动保存
- 无法解析章节编号：按修改时间排序
