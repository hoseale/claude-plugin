# 微信公众号文风指南

## 基础原则

### 原则一：读者至上

- 永远考虑读者为什么要读这篇文章
- 用"你"而非"用户"、"大家"来称呼读者
- 删除所有对读者没有价值的文字

### 原则二：简洁有力

- 能用短句就不用长句
- 能一段说清的不拆成三段
- 避免"的"字滥用（检查：的 的 的）

### 原则三：言之有物

- 每个观点都要有支撑（数据、案例、亲身经历）
- 避免空洞的废话（"随着时代的发展..."）
- 具体 > 抽象

---

## 标题规范

### 好标题的特征

- 简洁：不超过25个字
- 具体：让读者知道文章讲什么
- 有价值：读者看完想读
- 有悬念：引发好奇但不标题党

### 标题类型

| 类型   | 格式       | 示例                              |
| ------ | ---------- | --------------------------------- |
| 疑问式 | 抛出问题   | 《为什么年轻人不喜欢加班？》      |
| 数字式 | 数字+干货  | 《3个技巧，让你的文章阅读量翻倍》 |
| 对比式 | 反差吸睛   | 《月薪3000和30000的人，差在哪里》 |
| 热点式 | 借势热点   | 《ChatGPT爆火，我们该如何看待》   |
| 结论式 | 直接给答案 | 《职场新人最该避免的5个坑》       |
