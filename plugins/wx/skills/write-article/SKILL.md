# 技能：write-article

根据微信公众号账号主题和指定热点撰写 HTML 格式文章。

## 前置条件

- 已运行 `/wx.setup` 配置账号主题
- 已有待撰写的热点话题

## 执行流程

### 1. 确认写作需求

询问用户：

- **文章主题/热点**: 用户要写的具体话题
- **文章类型**: 资讯类 / 观点类 / 干货类 / 故事类
- **目标字数**: 如1500-2000字
- **特殊要求**: 如有

### 2. 加载上下文

读取以下文件：

- `account_profile.md` - 账号主题配置
- `style_guide.md` - 文风指南
- `article_template.md` - HTML 文章模板（包含完整结构和样式）

### 3. 分析热点

全网搜索热点的相关内容以及评论，了解读者关注点

### 4. 确定切入角度

- 如何将热点、读者关注点与账号核心话题结合
- 目标读者关心什么
- 如何体现账号独特视角
- 如果无法确定切入角度，可以不考虑账号定位，从普世价值切入

### 5. 撰写 HTML 文章

参考 `article_template.md` 的 HTML 模板生成文章。

**封面图处理**：

- 在封面图位置使用 HTML 注释标注配图建议
- 格式：`<!-- 封面图建议: {关键词描述} -->`

**文章结构要求**：

- **标题 (h1)**: 简洁有力，包含关键词
- **副标题**: 补充说明，可选
- **元信息**: 作者、日期、阅读时长
- **封面图**: HTML 注释标注配图关键词
- **摘要**: 50-100字核心内容概括
- **开头**: 用热点或痛点切入
- **主体**: 逻辑清晰，分点论述
- **结尾**: 总结 + 引导互动

**HTML 标签使用规范**：

- `h2` - 章节标题
- `p` - 段落
- `blockquote` 或 `div.quote` - 引用
- `div.key-point` - 重点提示
- `ul/ol` - 列表
- `strong` - 关键词强调
- `span.highlight` - 高亮

### 6. 生成封面和摘要

根据文章内容，生成：

- 封面图关键词描述（写入 HTML 注释）
- 文章摘要（用于微信预览）

### 7. 文章审查

- 检查内容
  - **标题引力**：能否吸引读者注意
  - **内容价值**：是否符合目标读者需求
  - **普世价值**：是否对大部分读者（包含目标读者）有价值
- 检查结果
  - 标题：不通过请重新撰写
  - 内容价值和普世价值：必须至少满足一个，如果两种都不满足，从步骤 4 开始重新构思文章

### 8. 保存文章

保存到 `articles/` 目录：

- 文件名: `article_YYYYMMDD_{话题关键词}.html`
- 包含完整 HTML 结构和样式

## 错误处理

- 账号配置不存在：提示先运行 `/wx.setup`
- 热点搜索失败：提示用户检查网络连接
- 缺少模板文件：使用默认 HTML 结构和样式
- 写作失败：保存草稿，提示用户手动完成
