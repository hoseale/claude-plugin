# 技能：setup

初始化微信公众号项目环境，配置账号主题。

## 执行流程

### 1. 验证环境

检查是否已存在项目结构。如存在，询问用户是否重新初始化。

### 2. 创建目录结构

```
wx-project/
├── account_profile.md    # 账号主题配置
├── article_template.md   # 文章模板文件
├── style_guide.md        # 内容风格文件
├── articles/             # 文章存档
└── trends/               # 热点记录
```

### 3. 引导用户配置账号主题

询问以下信息：

- **账号名称**: 微信公众号名称
- **所属领域**: 如：科技数码、财经投资、职场技能、生活方式、情感心理等
- **内容风格**: 如：专业深度、轻松有趣、幽默诙谐、严肃严谨
- **目标读者**: 如：25-35岁职场人群、大学生、创业者等
- **核心话题**: 3-5个核心内容方向
- **禁忌话题**: 不适合发布的内容方向
- **更新频率**: 如每周几篇

### 4. 生成 account_profile.md

从用户输入生成配置文件，包含以下内容：

- 账号领域
- 内容风格
- 核心话题
- 关键词库

文件保存到工作根目录 `account_profile.md`。

### 5. 复制模板文件

- `templates/article_template.md` 复制到工作根目录 `article_template.md`。
- `templates/style_guide.md` 复制到工作根目录 `style_guide.md`。

### 6. Git 初始化

如未初始化 git，运行 `git init` 并提交首次 commit。

## 错误处理

- 目录创建失败：通知用户并停止
- 文件写入失败：尝试备用方案
- git 已初始化：跳过 init，尝试提交
- 配置冲突：询问用户是覆盖还是保留现有配置
