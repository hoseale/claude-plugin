# 技能：analyze-trends

分析热点话题与账号主题的匹配程度，推荐值得蹭的热点。

## 执行流程

- [ ] Step1: 加载账号配置
- [ ] Step2: 加载热点数据
- [ ] Step3: 匹配度分析
- [ ] Step4: 选题推荐
- [ ] Step5: 保存分析结果

每个步骤必须独立运行。

### 1. 加载账号配置

读取工作目录下的 `account_profile.md`，获取：

- 账号领域
- 内容风格
- 核心话题
- 关键词库

### 2. 加载热点数据

读取工作目录 `trends/` 目录下的热点文件：

- 如有多个文件，默认读取最新的一个
- 或询问用户指定要分析的热点文件

### 3. 匹配度分析

对每个热点话题进行匹配度评估（0-100%）：

**匹配维度**：

- 主题相关性：热点与账号领域的关联程度
- 受众重合度：热点受众与目标读者的重叠程度
- 普世性：热点是否对所有读者都有价值
- 调性匹配度：热点风格与账号内容风格的契合度
- 竞争强度：同类内容竞争程度（低 = 机会大）

**权重计算**：

- 主题相关性：40%
- 受众重合度：20%
- 普世性：20%
- 调性匹配度：10%
- 竞争强度：10%

输出格式：

| 热点话题 | 匹配度 | 主题相关 | 受众重合 | 普世性 | 调性匹配 | 竞争强度 | 推荐指数 |
| -------- | ------ | -------- | -------- | ------ | -------- | -------- | -------- |
| xxx      | 85%    | 高       | 高       | 高     | 低       | 低       | ★★★★★    |
| xxx      | 60%    | 中       | 高       | 低     | 中       | 中       | ★★★☆☆    |

### 4. 选题推荐

- [ ] 对排名前 3 个热点话题进行深度搜索，获取详细内容。
- [ ] 对前 3 个热点话题进行主题创作。

**{热点话题}**

- 详细内容：[详细内容]
- 文章标题1：[文章标题及内容摘要]
- 文章标题2：[文章标题及内容摘要]
- 文章标题3：[文章标题及内容摘要]

### 5. 保存分析结果

将分析报告保存到工作目录 `trends/analysis_YYYYMMDD.md`

## 错误处理

- 账号配置不存在：提示用户先运行 `/wx.setup`
- 无热点数据：提示用户先运行 `/wx.check-trends`
- 分析失败：输出原始数据供用户手动参考
