# 技能：analyze-trends（爆款选题引擎）

分析热点话题与账号主题的匹配程度，并输出“可直接写作的选题方案”。

---

## 执行流程

- [ ] Step1: 加载账号配置
- [ ] Step2: 加载热点数据
- [ ] Step3: 分析热点数据
- [ ] Step4: 选题策略生成
- [ ] Step5: 保存分析结果

## 1. 加载账号配置

读取：

- 工作目录下的 `account_profile.md`

获取：

- 账号领域
- 内容风格
- 核心话题
- 关键词库

## 2. 加载热点数据

读取：

- `trends/` 目录下的最新文件

## 3. 分析热点数据

综合推荐 = 匹配度 × 50% + 爆款潜力 × 50%

### 1. 匹配度分析

维度：

- 主题相关性（40%）
- 受众重合度（20%）
- 普世性（20%）
- 调性匹配度（10%）
- 竞争强度（10%）

### 2. 爆款潜力评估

维度（0-5分）：

- 情绪强度（是否引发焦虑/愤怒/好奇）
- 冲突程度（是否存在对立观点）
- 反转空间（是否可以打破认知）
- 讨论度（是否容易引发评论）
- 新鲜度（是否刚发生）

## 4. 选题策略生成

对 Top 3 热点，输出：

### 【热点名称】

#### 1. 推荐切入角度

- 角度1：
- 角度2：
- 角度3：

#### 2. 推荐结构类型

- 冲突型 / 干货型 / 故事型（选1）

#### 3. 推荐“反转点”（必须）

示例：

- 你以为XXX，其实XXX

#### 4. 标题建议（3个）

---

#### 5. 写作提示（关键）

说明：

- 重点写什么
- 避免写什么
- 情绪方向（偏理性 / 偏情绪）

## 5. 保存分析结果

保存路径工作目录：`trends/analysis_YYYYMMDD.md`
