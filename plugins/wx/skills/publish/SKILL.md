# 技能：publish

**注意**: 此技能为预留接口，当前版本暂未实现。

## 计划功能

### 1. 草稿管理

- 查看本地草稿列表
- 编辑草稿内容
- 删除不需要的草稿

### 2. 微信公众号 API 集成

- 获取 Access Token
- 新建草稿箱内容
- 发布图文消息
- 定时发布

### 3. 发布预览

- 生成预览链接
- 发送到指定用户预览确认

## 当前状态

API 集成功能开发中，暂时仅提供本地文章管理能力。

## 使用建议

目前阶段，请手动将文章复制到微信公众号后台发布。
